== Description ==

Increase social sharing to boost your website trafficÂ with <strong>Wordpress Image Hover Lite</strong>. This plugin allows you to easily add social share icons on every images you've added via wordpress editor. 

<strong>Compatible with the brand new WordPress 4.2 :)</strong>

<h4>Features:</h4>
* <strong>Responsive</strong> - Desktop, laptop, tablet or mobile phone, WP Image Hover will adjust and look good on it
* <strong>Alignment Support</strong> - None, center, right and left images alignment support!
* <strong>8 Socia Media Sharing Icon</strong> - Supports facebook, twitter,linkedin, pinterest, google+, digg digg, stumbleupon and delicious social sharing.
* <strong>TinyMCE Toolbar Plugin</strong> - Easily add and configure share message and social icon visibility via user friendly editor icon!
* <strong>Browsers Compatible</strong> - Tested on every browsers, including IE8.
* <strong>Powerful Admin Options</strong> - Easy and intuitive admin options for you to easily set default options.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. That's it :) , additional wordpress menu "WP Image Hover Lite" is now added
